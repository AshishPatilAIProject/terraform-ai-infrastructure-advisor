Remediation Plan

Finding: S3 Bucket Missing Encryption
Risk: Data stored in the bucket is vulnerable to unauthorized access and breaches due to lack of encryption.
Terraform Fix:
```hcl
resource "aws_s3_bucket" "example" {
bucket = "my-bucket"
server_side_encryption_configuration {
rule {
apply_server_side_encryption_by_default {
sse_algorithm = "AES256"
}
}
}
}
```

Finding: Open SSH Security Group
Risk: Allows unrestricted SSH access from the internet, exposing instances to brute force attacks and unauthorized access.
Terraform Fix:
```hcl
resource "aws_security_group" "ssh_restrict" {
name = "ssh_restrict"
description = "Allow SSH access from known IPs only"
vpc_id = var.vpc_id

ingress {
from_port = 22
to_port = 22
protocol = "tcp"
cidr_blocks = ["203.0.113.0/24"] # Replace with trusted IP range
}

egress {
from_port = 0
to_port = 0
protocol = "-1"
cidr_blocks = ["0.0.0.0/0"]
}
}
```
